## Open locally

Open [index.html](/Users/derrick/Desktop/Resume/portfolio/index.html) in a browser, or serve the folder with a simple static file server.

## What to publish

Publish the entire `portfolio/` folder, including:

- `index.html`
- `styles.css`
- `Derrick_Karake_Resume.pdf`

## LinkedIn note

LinkedIn does not host standalone websites. The practical workflow is:

1. Publish this folder on GitHub Pages, Netlify, or Vercel.
2. Copy the public URL.
3. Add that URL to your LinkedIn `Featured` section, `Contact info`, or `About` section.
